# 🌲 Floresta Encantada

É um jogo em 2D desenvolvido em Python.  
Você controla um **mago** que deve coletar moedas que caem do céu para ganhar pontos.  
Mas cuidado: **espadas** também caem, e se atingirem o mago, o jogo acaba.

---

## 🎮 Como jogar

- Use as **setas do teclado** para mover o mago pela tela.  
- Pegue as **moedas** para aumentar sua pontuação.  
- Evite as **espadas**, pois elas encerram a partida.
